Gemini AI Chatbot using n8n
This is a beginner AI Chatbot workflow created using n8n and Google Gemini.
Features:
AI Chat Workflow
Google Gemini Integration
Memory enables conversations
Simple AI agent setup
Beginner friendly project

Technologies used:
n8n
Google Gemini API

The workflow:
Receives user chat messages 
Sends input to AI Agent
Uses Gemini Model for responses
Maintans memory context

Setup
1)Import workflow.json into n8n
2)Add your Gemini API credentials
3)Activate workflow
4)Start chatting
Note:This project was created for learning AI automation using n8n.
